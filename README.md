# 专利项管理系统
基于nw.js桌面技术开发的专利管理系统桌面应用。兼容到Window XP系统。出于兼容性考虑没有使用Electron.js来开发

## 主要功能

- 导入Excel专利数据文件到系统中，解析excel数据并存储专利数据到本地数据库(linvodb3)中
- 批量导入图片。关联所有的图片到数据库对应的专利项中
- 支持专利项在系统中通过丰富的查询搜索方式寻找并展示出专利项
- 支持根据专利项的各个指标进行统计和统计数据可视化
- 支持丰富的单项专利项的展示
- 支持对专利项定制性对导出
